Create a watchlist using the File -> New -> Watchlist menu. The watchlist wizard allows you to define the name of the watchlist and the columns to display. To add the securities to a watchlist drag a security from the securities view and drop it to the watchlist. The security will be added at the end of list. Use the Settings menu to reorder the list.

Watchlists can layout the securities using a table view or a ribbon view.

Right click on a security in the table view to open a pop-up menu that allows you to perform various operations on the security, such as opening the level II data view or the historical chart. The pop-up menu is not available in the ribbon view.
