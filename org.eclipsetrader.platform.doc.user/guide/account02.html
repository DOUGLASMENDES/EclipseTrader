Creating a transaction

To create a transaction:

  1. Right-click on an account.
  2. From the popup-menu, select New Transaction.
  3. In the Date / Time field, type the date and time of the transaction (defaults to the current date and time).
  4. Select the account (defaults to the selected account).
  5. Select the security.
  6. Select Buy or Sell operation.
  7. In the Quantity field, type the number of shares to buy or sell.
  8. In the Price field, type the price paid for each shares.
  9. In the Minimum field, type the minimum commissions amount to pay for each transaction.
  10. In the Expenses field, type the total commissions amount paid for the transaction.
  11. Click OK.

The new transaction is listed in the account's transactions view, and the portfolio view is updated to reflect the changes.

Tip: Drag a security from the securities or watchlist views to the account or transaction view to enter a transaction and automatically fill all fields.
