Creating a security

To create a security:

  1. On the main menu bar, click File > New > Stock. The New Stock wizard opens.
  2. In the Security name field, type the name of the security (the company's name for example), and select the currency.
  3. Click Next to setup the feed symbols.
  4. In the Identifier name field type the symbol with which the security is quoted on the market. In the properties table you can set a connector-specific symbol. The symbol defined above is used as the default symbol for all connectors.
  5. Clic Next to select the market.
  6. In the markets table select the market on which the security is quoted. The market defines the default settings for quote and historical backfill sources.
  7. Click Finish. The new security is listed in the securities view.
