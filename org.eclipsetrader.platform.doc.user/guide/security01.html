Creating a security

To create a security:

  1. On the main menu bar, click File > New > Security. The New Security wizard opens.
  2. In the Code field, type the unique code for the security, In the Description field, type a description for your new security (the company's name for example), and select the (optional) currency.
  3. Click Next to setup the realtime quote feeds.
  4. Select the source that provides the realtime quotes for the new security.
  5. If available, select the Exchange name.
  6. In the Symbol fields, type the symbol used by the data provider to identify the security. If these fields are left blank, the Code field from the first page is used.
  7. (Optional) Click Next to setup the level II quote feeds.
  8. Select the source that provides the level II view for the new security.
  9. (Optional) Click Next to setup the historical chart feed.
  10. Select the source that provides the historical chart data for the new security.
  11. (Optional) Click Next to setup the intraday chart build options.
  12. Select Enable automatic build if you want that EclipseTrader builds the intradat charts automatically while downloading the realtime quotes. If disabled, the intraday charts are provided by the historical charts data source, if available.
  13. Set the begin and end time, and the days of the week to collect the data.
  14. Set how many days you want to keep in the intraday charts in the Keep (days) field.
  15. Click Finish. The new security is listed in the securities view.

Tip: The data feed plugins can provide additional wizards with a list of the available securities that do not require to enter all data manually. These wizards appears in the File > New menu on the main menu bar.
