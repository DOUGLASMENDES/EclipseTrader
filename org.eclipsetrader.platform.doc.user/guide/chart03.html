Add a drawing object

To add a drawing object:

  1. Open the view's menu, select New Object.
  2. From the menu, select the object you want to add.
  3. Click on the chart area to position the object, and drag, if necessary, to resize the indicator.

The move an object, click on the object and drag the mouse.

To resize an object, click on one of the corners and drag the mouse.
