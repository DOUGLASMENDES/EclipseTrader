Add an indicator

To add an indicator:

  1. Right-click on the chart area.
  2. From the pop-up menu, select New Indicator, the New Indicator wizard opens.
  3. Select the indicator from the list.
  4. Click Next.
  5. Select where the new indicator should be placed.
  6. Click Next.
  7. Change the indicator's parameters.
  8. Click Finish.

The indicator is shown.
