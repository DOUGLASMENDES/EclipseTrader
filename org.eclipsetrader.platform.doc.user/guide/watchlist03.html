Searching price patterns

To search for a price patter:

  1. On the view's tools bar, open the pull-down menu and click Search Pattern.
  2. Select the Pattern to search, or All to seach all available patterns.
  3. Select the Begin and End dates to search, or leave the fields blank to search all available dates.
  4. Select the Period (daily, weekly, monthly) to search.
  5. Select the number of Bars.
  6. Select Bullish Only if you want to seach only bullish patterns.
  7. Select Search all occurrences if you want to seach all occurrences within the selected historical data. If unselected, only the last occurrence is searched.
  8. Click OK.

The search results are shown in the Pattern Seach view.

Tip: Right-click on a watchlist item to search the patterns on the item's security.
