Creating a watchlist

To create a watchlist:

  1. On the main menu bar, click File > New > Watchlist. The New Watchlist wizard opens.
  2. In the Name field, type the name of your new watchlist, and select the (optional) currency.
  3. Click Next. Select the columns to show in the watchlist. Reorder the columns using the up and down buttons next to the Shown columns list.
  4. (Optional) Click Next. Select the securities to show in the watchlists. Reorder the securities using the up and down buttons next to the Shown securities list
  5. Click Finish.

The watchlist opens in the workbench. To open an existing watchlist, on the main menu bar, click Window > Open Watchlist > the name of the watchlist to open.

The securities can be added to the watchlist by dragging the security from the securities view to the watchlist.
