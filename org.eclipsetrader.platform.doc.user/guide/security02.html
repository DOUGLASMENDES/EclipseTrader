Creating a watchlist

To create a watchlist:

  1. On the main menu bar, click File > New > Watchlist. The New Watchlist wizard opens.
  2. In the Name field, type the name of your new watchlist.
  3. Click Next. Select the columns to show in the watchlist. Reorder the columns using the up and down buttons next to the Shown columns list.
  4. Click Finish.

The watchlist opens in the workbench. To open an existing watchlist, open the Navigator view, expand the Watchlist folder and double click on the name of the watchlist to open.

The securities can be added to the watchlist by dragging the security from the navigator view.
