Using the portfolio

To open the portfolio view:

  1. On the main menu bar, click Window > Show View > Portfolio.

The portfolio view reflects the accounts groups organization. The current open positions are shown for each account.

Tip: Drag a security from the securities or watchlist views to an account on the portfolio view to enter a new transaction and automatically fill all fields.

To close a position:

  1. Right click on the position to close.
  2. From the pop-up menu, select Close Position.
  3. The transaction dialog opens.
  4. Change the field values as required.
  5. Click OK.

The portfolio and accounts views are updated to reflect the change.
