Simulation

To simulate the results of a trading strategy:

  1. Open the Strategy Explorer view, if not already open.
  2. Right-click on a stratgy line.
  3. From the pop-up menu, select Run Simulation.... The simulation period dialog opens.
  4. Select the period to test.
  5. Click OK.

The test is run in the background and the results are shown in the report view. The view will show the account balance over the strategy run period and a list of all trades.
