Using the alerts

To create an alert:

  1. Right-click on a watchlist row.
  2. From the pop-up menu, select Properties.
  3. Select the Alerts page.
  4. Click on the bell icon, the new alert wizard opens.
  5. Select the alert to add and fill the wizard pages.
  6. Click Finish the new alert is added to the list. Click Ok to exit the properties dialog.

The alert is added to the selected security.

The alerts are checked whenever there is an update in the security's quote. The alert's match is signaled with a pop-up dialog and an highlight of the watchlist's row..

To edit an alert:

  1. Right-click on the watchlist row.
  2. From the pop-up menu, select Properties.
  3. Select the Alerts page.
  4. Select the alert to edit from the alerts table.
  5. Change the parameters.
  6. Click OK.
