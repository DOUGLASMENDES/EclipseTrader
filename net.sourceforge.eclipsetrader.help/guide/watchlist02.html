Using the alerts

To create an alert:

  1. Right-click on the security.
  2. From the pop-up menu, select New Alert. The New Alert wizard opens.
  3. Select the Alert type that you want to create.
  4. Click Next.
  5. Set the parameters specific to the selected alert type.
  6. Click Finish.

The alert is added to the selected watchlist item.

The alerts are checked whenever there is an update in the security's quote. The alert's match is signaled in the events view. To open the events view, on the main menu bar, click Window > Show View > Events.

To edit an alert:

  1. On the view's tools bar, open the pull-down menu and click Settings.
  2. Expand the Securities page.
  3. Select the security.
  4. Select the alert from the alerts table.
  5. Change the parameters.
  6. Click OK.

The alerts table shows a short description of the alert and the date and time when the last match was found.
