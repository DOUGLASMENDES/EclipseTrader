Creating a security

To create a security:

  1. On the main menu bar, click File > New > Security. The New Security wizard opens.
  2. In the Code field, type the unique code for the security, In the Description field, type a description for your new security (the company's name for example), and select the currency.
  3. Click Next to setup the realtime quote feeds.
  4. Select the sources that are providing the realtime quotes and (optional) level II view for the new security.
  5. If available, select the Exchange name.
  6. In the Symbol fields, type the symbol used by the data provider to identify the security. If these fields are left blank, the Code field from the previous page is used.
  7. (Optional) Click Next to setup the historical chart feed.
  8. Select the sources that are providing the historical chart data for the new security.
  9. If available, Select the Exchange name.
  10. In the Symbol fields, type the symbol used by the data provider to identify the security. If these fields are left blank, the Code field from the first page is used.
  11. Click Finish. The new security is listed in the securities view.

Tip: The data feed plugins can provide additional wizards with a list of the available securities that do not require to enter all data manually. These wizards appears in the File > New menu on the main menu bar.
