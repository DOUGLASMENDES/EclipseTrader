Deleting accounts

To delete an account:

  1. Select the account that you want to delete.
  2. Press the Delete key
  3. Click Yes in the dialog which appears.

Tip: You can achieve the same result by selecting Delete from the pop-up menu.

To delete a group:

  1. Select the group that you want to delete.
  2. Press the Delete key
  3. Click Yes in the dialog which appears.

Tip: You can achieve the same result by selecting Delete from the pop-up menu.

Warning! Deleting a group also deletes all contained accounts and groups.
