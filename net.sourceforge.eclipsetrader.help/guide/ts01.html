Creating a trading system

To create a trading system:

  1. On the main menu bar, click File > New > Trading System. The New Trading System wizard opens.
  2. Select one of the available trading systems.
  3. Click Next.
  4. Select the Account to use.
  5. Select the Security that is traded with the new system.
  6. In the Max. Exposure field, type the maximum value of the security in the portfolio. Zero for no limits.
  7. In the Min. Trade Amount field, type minimum amount for a trade.
  8. In the Max. Trade Amount field, type maximum amount for a trade. Zero for no limits.
  9. Click Next.
  10. Set the parameters related to the selected trading system.
  11. Click Finish. The new trading system is listed in the trading systems view.
