Installazione di nuove funzioni con il Gestore aggiornamenti

Per individuare e installare una nuova funzione all'interno di un prodotto (generalmente è necessario l'accesso alla rete):

  1. Aprire l'installazione guidata selezionando Guida > Aggiornamenti software > Trova e installa. Viene visualizzata la procedura guidata.
  2. Selezionare il secondo pulsante, "Cerca nuove funzioni da installare" e premere Avanti.
  3. Creare un segnalibro per un sito di aggiornamenti in cui vengono pubblicati le funzioni ed i plugin di Eclipse. Nell'elenco dei siti in cui effettuare le ricerche, selezionare Aggiungi  Sito aggiornamenti per aggiungere un sito remoto, o Aggiungi sito locale se il sito è disponibile su un'unità locale (incluso un CD) oppure Aggiungi sito archiviato, quando il sito è disponibile localmente ma è impacchettato come file jar o zip.
  4. Nella finestra di dialogo per l'aggiunta di un sito, fornire un nome al sito, ad esempio "CompanyA" ed immettere l'URL, ad es. "http://companyA.example.com/eclipseupdates".
  5. Dopo aver aggiunto il sito, espanderlo alle categorie delle versioni delle funzioni disponibili a quel sito di aggiornamento. Questa azione consente di connettersi al sito Web per scoprire quali funzioni sono disponibili.
  6. Selezionare le categorie in cui effettuare la ricerca e premere Avanti.
  7. Attendere il termine della ricerca e selezionare le funzioni da aggiungere. Per visualizzare la descrizione o le proprietà più dettagliate di una funzione, selezionare la funzione e premere il pulsante "Proprietà".
  8. Una volta scelte le funzioni da installare, selezionare Avanti.
  9. Leggere attentamente gli accordi di licenza relativi alle funzioni. Se le condizioni di tutti questi contratti sono accettabili, scegliere "Accetto i termini del contratto di licenza". Non continuare nel download delle funzioni se le condizioni dei contratti non sono accettabili.
  10. Se una delle funzioni selezionate per l'installazione contiene funzioni facoltative, verrà visualizzata una pagina in cui sarà possibile scegliere se installare o meno tali funzioni. Generalmente, le funzioni facoltative non sono essenziali per l'attività della funzione principale.
  11. La pagina Percorso di installazione controlla il percorso sul computer locale in cui devono essere installati i file della nuova funzione. Selezionare la directory nella quale il prodotto è installato e scegliere Avanti. Se il prodotto è installato in una directory per cui non si dispone di accesso in scrittura, è necessario contattare l'amministratore di sistema per ottenere l'installazione della funzione in modo che sia generalmente disponibile. L'altra opzione è di premere Aggiungi e selezionare una directory per la quale si dispone dell'accesso in scrittura.
  12. Le versioni di funzione possono presentare la firma digitale dall'azienda che le fornisce. Ciò consente di verificare in modo più semplice che le funzioni e i plugin che stanno per essere scaricati e installati provengano da un fornitore affidabile.


    Attenzione: a causa della possibilità di plugin che possono causare problemi o essere in altro modo dannosi, è necessario scaricare solo le parti affidabili delle funzioni.


    Selezionare Installa per continuare il download e l'installazione.

  13. Una volta completato il download della nuova funzione e dei plugin e installati i relativi file nel prodotto che si trova sul computer locale, viene elaborata una nuova configurazione che incorpora tali funzioni e plugin. Fare clic su Sì quando viene richiesto di uscire e riavviare il workbench per rendere effettive le modifiche. Per aggiungere altre nuove funzioni prima di riavviare, fare clic su No e ripetere l'operazione. Se si tratta di una nuova funzione e può essere installata dinamicamente nella configurazione corrente, verrà visualizzato anche un pulsante "Applica ora". Questo non riavvierà la piattaforma ma configura la funzione ed i plugin nella configurazione corrente.
